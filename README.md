# 一维带电小球碰撞仿真｜迭代镜像法 替代点电荷近似
## 项目概述
本项目基于VPython实现一维双导体边界内异种带电小球碰撞可视化仿真，核心创新：
1. 使用**迭代镜像法**计算边界感应电荷的库仑力与电势能，替代传统忽略边界的点电荷模型；
2. 采用超高阶镜像（100阶）作为理论真值，计算有限阶镜像的能量截断误差，定量评估镜像法可行性；
3. 完整记录小球运动、受力、动能、电势能数据，独立绘图脚本输出5张分析图，输出误差统计报表；
4. 支持小球碰撞电荷均分交换、实时可视化面板、自定义物理参数。

## 参数配置说明

### ball_data.py

小球固定初始参数，仿真启动前可在可视化界面二次修改：

- BALL_A_MASS / BALL_B_MASS：两球质量 (kg)
- BALL_A_POS / BALL_B_POS：初始坐标 (m)
- BALL_A_VEL / BALL_B_VEL：初始速度 (m/s)
- BALL_A_CHARGE / BALL_B_CHARGE：小球电荷量 (C)

### record_config.py（核心算法参数）

```
MIRROR_ITER_ORDER = 8      # 你使用的迭代镜像阶数（仿真计算用）
THEORY_MIRROR_ORDER = 100  # 理论真值超高阶镜像（误差计算基准）
RECORD_INTERVAL = 0.2      # 数据存储时间间隔(s)
DECIMAL_PLACES = 3         # 普通数值保留小数
ERR_DECIMAL = 4            # 误差数值保留小数
```

- 提高 MIRROR_ITER_ORDER 可降低截断误差，但仿真速度变慢；
- 误差判定标准：
  1. 平均相对误差 < 1%：精度优秀，完全替代点电荷模型
  2. 1% ~ 5%：满足课程、教学仿真使用
  3. 5%：镜像阶数不足，需增大迭代层数

## 核心算法原理

1. **传统点电荷模型缺陷**：忽略左右导体边界感应电荷，电势能、受力计算存在巨大系统误差，能量不守恒；
2. **迭代镜像法**：通过不断镜像边界电荷，等效替代导体表面感应电荷，取有限阶数近似无穷镜像系统；
3. **误差计算逻辑**： 阶阶 δ_E 为总机械能相对截断误差，仅反映镜像截断带来的算法偏差。
